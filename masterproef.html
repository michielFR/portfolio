Masterproef

  • UGent
  • MongoDB, JavaScript

De master opleiding gaat gepaard met een masterproef. Ik koos voor een samenwerking met een onderzoeksgroep van UGent zelf. Deze onderzoeksgroep is gevestigd in het IGent gebouw op de technologie campus te Zwijnaarde. Dit gebouw was meteen ook het vertrekpunt voor mijn masterproef. In het gebouw zijn een aantal verdiepingen waar Bluetooth nodes aanwezig zijn. De nodes vormen een Bluetooth Mesh netwerk. Dit netwerk kan gemonitord worden met behulp van een digital twin.

Mijn masterproef had de opzet om data afkomstig van de digital twin op te slaan op een gestructureerde en uitbreidbare manier. Deze data werd vervolgens gebruikt om dashboard op te stellen waarmee data kon worden voorgesteld en hier nuttige conclusies uit konden worden gehaald.

uiteindelijk was er een setup met een MongoDB databank, een API die er data naar kon schrijven maar vooral uit ophalen met een reeks gedefinieerde endpoints. De calls werden gedaan vanuit een Grafana dashboard. Voor zo'n Grafana dashboard werd een eigen plug-in geschreven die met de API sprak en data aanbood aan het dashboard.
